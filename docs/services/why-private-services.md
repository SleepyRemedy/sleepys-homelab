# Why Private Services

## Goal

All services in RemingtonLab are private by default.

## Security Philosophy

Services should only be accessible through Tailscale unless there is a specific reason to expose them publicly.

## Protected Services

- Proxmox
- Homepage Dashboard
- Grafana
- Prometheus

## Public Exceptions

Examples of services that may be exposed in the future:

- Personal Website
- Portfolio
- Minecraft Server

## Notes

This decision reduces attack surface and simplifies security management.
